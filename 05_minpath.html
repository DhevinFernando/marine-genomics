in order to isolate the unique ID and EC number of each gene we will transform the tsv using an awk command ,-F ’ is included to indicate the file is tab seperated

head ../prokka/contig_115/contig_115.tsv | awk -F '\t' '{print $1,$5}'

we will then further filter this output so that only entries with an EC_number remain

head ../prokka/contig_115/contig_115.tsv | awk -F '\t' '$5{print $1,$5}'

we now remove the header line and ensure the file is still tab seperated with

head ../prokka/contig_115/contig_115.tsv | awk -F '\t' '$5{OFS="\t";print $1,$5}' | grep -v '^locus'

since the previous code only applies to the first 10 lines lets change it to cat instead of head and make it act on all the genes in the file

cat ../prokka/contig_115/contig_115.tsv | awk -F '\t' '$5{OFS="\t";print $1,$5}' | grep -v '^locus'

we then send this output to file to be used an input in minpath

cat ../prokka/contig_115/contig_115.tsv | awk -F '\t' '$5{OFS="\t";print $1,$5}' | grep -v '^locus'> contig_115.ec

we then run minpath on this file

minpath -ec contig_115.ec -report contig_115.ec.report -details contig_115.ec.details

lets view the first 5 lines of contig_115.ec.report

head contig_115.ec.report

Now using our completed prokka output we can run minpath for all bins

lets first prepare the minpath input for all bins

while read bin;do 
  cat ../prokka/$bin/${bin}.tsv | awk -F '\t' '$5{printf("%s\t%s\n", $1,$5)}' | grep -v '^locus' > ${bin}.ec 
done < ../checkm/bbd_topbins.txt

now we run minpath on every bin

while read bin;do 
  minpath -ec ${bin}.ec -report ${bin}.ec.report -details ${bin}.ec.details
done < ../checkm/bbd_topbins.txt
